Route 组件总结

这节课的主要内容是，对于 Route 组件，如果你希望向 router 将渲染的特定组件传递属性，你需要使用 Route 的 render 属性。正如你所看到的， render 使你能够控制渲染组件，进而使你能够向要渲染的组件传递任何属性。

总之， Route 组件是使用 React Router 构建应用的关键部分，因为该组件将根据当前的 URL 路径决定要渲染哪些组件。
