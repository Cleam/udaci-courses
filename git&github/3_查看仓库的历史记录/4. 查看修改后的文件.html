git log --stat 简介

The git log 命令有一个选项可以用来显示 commit 中更改的文件以及添加或删除的行数。该选项为 --stat（stat 是“统计信息 statistics”的简称）：

$ git log --stat

git log --stat 小结

总结下， --stat 选项用于更改 git log 显示信息的方式：

$ git log --stat

此命令会：

  • 显示被修改的文件

  • 显示添加/删除的行数

  • 显示一个摘要，其中包含修改/删除的总文件数和总行数
